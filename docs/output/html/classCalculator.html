Подробное описание

Класс для вычисления значений ветвящейся функции и формирования строки данных графика.

Функция задана по трём ветвям:

  • f(x) = |x·a| − 2, при x < −2
  • f(x) = b·x² + x + 1, при −2 ≤ x < 2
  • f(x) = |x − 2| + c, при x ≥ 2
