ТИМП — Подгруппа 5

Qt-приложение с клиент-серверной архитектурой. Авторизация, менеджер задач, графики функций — всё через TCP + JSON-протокол поверх SQLite.

Стек: C++ / Qt5 / SQLite
Порт: 33333
Docker: rainkord/timp-server
Тесты: Qt Test

🖥 Клиент

Qt Widgets-приложение. Авторизация, регистрация с email-верификацией, менеджер задач, редактор и отрисовка графов.

⚙️ Сервер

TCP-сервер на Qt. Маршрутизирует JSON-запросы, работает с SQLite через Database singleton, отправляет письма через Gmail SMTP.

📦 Docker

Образ Ubuntu 22.04. Сам собирает проект и запускает echoServer. Готовый образ: rainkord/timp-server:latest.

🧪 Тесты

Qt Test для модуля calculator — вычисление f(x) и генерация точек графика.

Быстрый старт

# Запустить сервер через Docker (самый простой способ)
docker pull rainkord/timp-server:latest
docker run -d --name timp-server -p 33333:33333 rainkord/timp-server:latest

# Собрать и запустить клиент локально
cd client
qmake client.pro
make -j$(nproc)
../build/client

Email-конфигурация

Настройка server/email.txt для отправки писем с кодами подтверждения

Файл email.txt добавлен в .gitignore — никогда не коммитьте его в репозиторий!

Формат файла

# Ключ можно получить: https://myaccount.google.com/apppasswords
email=your_address@gmail.com
key=xxxx xxxx xxxx xxxx

Как получить пароль приложения Google

  1. Перейдите на myaccount.google.com/apppasswords
  2. Войдите в аккаунт Google, от которого будет работать рассылка
  3. В поле «Название приложения» введите любое имя (например, TIMP Server)
  4. Нажмите Создать — Google выдаст 16-символьный пароль вида xxxx xxxx xxxx xxxx
  5. Скопируйте его в поле key= файла email.txt

Готовый пример

# Ключ можно получить: https://myaccount.google.com/apppasswords
email=example@gmail.com
key=abcd efgh ijkl mnop

Docker

Сервер упакован в образ Ubuntu 22.04. Клиент собирается локально.

Перед сборкой образа обязательно заполните server/email.txt. Dockerfile копирует его внутрь образа.

Быстрый старт (готовый образ)

# Скачать готовый образ с Docker Hub
docker pull rainkord/timp-server:latest

# Запустить контейнер
docker run -d --name timp-server -p 33333:33333 rainkord/timp-server:latest

# Проверить логи
docker logs -f timp-server

Сборка из исходников

# Из корня репозитория
docker build -f docker/Dockerfile -t timp-server .
docker run -d --name timp-server -p 33333:33333 timp-server
